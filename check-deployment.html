Kiểm tra file đang được serve

Nếu bạn thấy file này, có nghĩa là GitHub Pages đang serve từ root repository.

Bạn cần cấu hình GitHub Pages để serve từ thư mục dist/ hoặc branch gh-pages.

Hướng dẫn:

  1. Vào Settings → Pages trong repository GitHub
  2. Chọn "GitHub Actions" làm source (nếu đã có workflow)
  3. HOẶC tạo branch gh-pages và copy nội dung dist/ lên đó
  4. Xem file README_DEPLOY.md để biết chi tiết
